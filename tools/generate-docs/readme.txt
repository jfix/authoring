run-all-stages.xpl requires two parameters:

output-folder - the location in which the docs should be created. This should be a folder and should include the trailing /.
version - a version number.

An input port "supplements" can take any number of docbook files.

The output folder described above should contain a folder "help" containing the four gif files included.
In addition, the folder should contain a folder resources/stylesheets containing the style.css file included.
