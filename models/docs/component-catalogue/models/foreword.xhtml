    • <foreword> - used by Narrdoc.

    The following definition of <foreword> is used by the Narrdoc model.
    This version of the <foreword> element is applicable when used as a child to the following elements:
    • <frontmatter>
    • <part>

    Elements and Attributes available within <foreword>

    PAC Documentation

    Container used to begin a foreword section. It store the text converted from the Word style Foreword Heading.<br> foreword element can be used in the following sections: <ul><li>frontmatter</li> <li>part</li></ul>
