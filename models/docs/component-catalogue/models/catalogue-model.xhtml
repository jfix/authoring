    The Component Catalogue model uses the namespace

    The root element is <document>.

    This model is a comprehensive catalogue of all the components used in narrative models for capturing official OECD documents and publications. To date, it supports all documents and camera-ready and typset publications.

    Help is available (opens in a separate window or tab).
